

# 玩机盒

## 介绍

玩机盒是一款可以快速激活各种应用程序的工具。通过简单的交互式界面，用户可以输入序号来选择并激活不同的应用。

## 功能特点

- 快速激活多种应用程序
- 简洁的交互式界面
- 易于使用，无需复杂配置

## 使用方法

1. 确保已安装 Python 环境
2. 运行程序：
   ```bash
   python wanji.py
   ```
3. 根据界面提示，输入序号进行交互操作

## 依赖环境

- Python 3.x
